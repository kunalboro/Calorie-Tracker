# Calorie Tracker (Machine Learning Project)

A machine learning-based application that predicts **calories burned during exercise** using user input data, with an interactive desktop GUI.

---

## Overview

This project uses a **regression-based machine learning model** to estimate calories burned based on exercise-related parameters.  
It also includes a **Tkinter-based GUI**, allowing users to input data and get real-time predictions.

---

## Tech Stack

- **Programming Language:** Python  
- **Machine Learning:** Scikit-learn  
- **GUI:** Tkinter  
- **Libraries:** NumPy, Pandas, Matplotlib  

---

## Features

- Data preprocessing and feature engineering  
- Regression model for prediction  
- Interactive GUI using Tkinter  
- Real-time calorie prediction  

---

## Model Performance

- Achieved **R² Score: 0.99**  
- High prediction accuracy on test data
